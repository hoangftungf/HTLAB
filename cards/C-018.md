# C-018 — Rebuild My Blocks: dynamic define block + multi-param modal + Edit function context menu

status: done
deps: C-017
implements: FR7

## Scope

Rebuild My Blocks từ đầu để khớp hoàn toàn với giao diện Whalesbot Block Studio:

1. **Dynamic mutator cho `my_block_definition`**: define block dùng mutator để tạo tham số inline sau tên hàm. Tên hàm sinh random suffix như `name_XXXX`. Không giới hạn số lượng tham số.
2. **Dynamic call blocks** (`my_block_call_statement`, `my_block_call_value`): dùng mutator để sync với số arg tương ứng với define block.
3. **Modal "Create a function - HTLABS block"**: đổi thương hiệu từ "Whalesbot scratch" → "HTLABS block". Cho phép thêm nhiều tham số (number, boolean) và label. Tham số number hiện dạng pill màu trắng trong preview. Tham số boolean hiện dạng lục giác màu trắng. Bỏ giới hạn 1 tham số.
4. **Right-click "Edit function"**: Khi right-click vào define block → context menu có option "Edit function" → mở lại modal "Edit function - HTLABS block" với trạng thái hiện tại.
5. **Text label hoạt động thực sự**: Thêm text label tạo label element trong preview.
6. **Generator IR update**: `functionDefinitionFromBlock` đọc params từ mutator (`params_` array) với backward compat fallback cho saves cũ.
