## Scope

Wire everything together, verify the full MVP flow end-to-end, polish rough edges, and deploy to a public URL.

1. **Full integration** — Blockly editor -> IR generator -> Interpreter -> SimulationCore -> PixiJS renderer -> Telemetry panel, all connected through Zustand store
2. **Sample map** — create or source 1 EnjoyAI-style sample map (PNG + JSON metadata) with a simple line course
3. **Sample program** — pre-loaded Blockly program that demonstrates basic line following on the sample map
4. **Error handling** — show user-friendly errors for: invalid IR, sensor calibration not done, execution timeout, map load failure
5. **Loading states** — map loading spinner, IR generation feedback
6. **Polish** — layout refinements, responsive sidebar, consistent font/color, favicon, page title
7. **Deploy** — Vercel static site deploy, configure custom domain or use `.vercel.app` URL
8. **Smoke test** — end-to-end: open deployed URL -> load sample map -> run sample program -> robot follows line -> telemetry shows live data
