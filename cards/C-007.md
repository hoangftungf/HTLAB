# C-007 — Blockly custom blocks + IR codegen + save/load

status: done
deps: C-003, C-005
implements: FR7, FR3 (codegen), FR8

## Scope

Build the Blockly editor with custom blocks for EnjoyAI Robotics programming, the IR code generator, and project save/load to localStorage.

1. **Custom Blockly blocks (~13 blocks)** — each with custom UI (color, icon, inputs), toolbox categories:
   - Hardware: `initialize`, `calibrate_grayscale`
   - Movement: `patrol_line`, `turn_left`, `turn_right`, `start_motor`
   - Sensors: `read_sensor_road`, `sensor_group_detected`, `line_position`
   - Logic: `if_sensor`, `repeat_loop`, `wait_block`
   - Variables: `set_var`
2. **IR code generator** — each block type has a generator function that outputs `IRCommand[]`. Full workspace → `IRProgram` JSON.
3. **Blockly workspace config** — custom toolbox JSON, Zelos renderer, zoom/scroll/trashcan
4. **Integration with app** — Blockly workspace in left panel, "Generate IR" button, IR JSON shown in status bar, "Run Program" feeds to Interpreter
5. **Project save** — save Blockly workspace XML + IR program to localStorage
6. **Project load** — list saved projects, load one, restore Blockly workspace
