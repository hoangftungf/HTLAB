# C-006 — Telemetry panel + app integration

status: done
deps: C-004, C-005
implements: FR5 (panel), FR6 (replay UI)

## Scope

Add the telemetry display panel, Zustand state management, and wire SimulationCore + renderer + controls together into a working interactive app.

1. **Zustand store** — `useSimStore` for simulation state (running, speed, tick, replay mode), subscribed by all UI components
2. **Simulation controls** — Run/Pause/Reset/Step buttons with keyboard shortcuts (Space=toggle, R=reset, S=step)
3. **Speed control** — slider 0.25x -> 4x (controls ticks per render frame)
4. **Telemetry panel** — sidebar showing:
   - 5 sensor road values (Road 1->5) with bar visualization (color = intensity 0-100)
   - Motor speeds (left/right) with bars
   - Pattern text display (large mono text: `00100`)
   - Line position number (-100 to +100) with visual gauge
   - Current tick counter
5. **Replay controls** — Step forward/backward, speed, jump to tick
6. **App wiring** — connect C-001 SimState -> PixiJS renderer -> controls -> telemetry panel through Zustand
