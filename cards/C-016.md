# C-016 - Overlay Blockly editor over simulation map

status: done
deps: C-015
implements: FR7, FR8 (toolbox usability and responsive layout polish)

## Scope

Move the Blockly editor out of the fixed three-panel layout and make it an overlay on top of the simulation map.

1. The simulation/map remains the primary full-width work surface.
2. The Blockly editor opens as a wide overlay so toolbox/flyout blocks have substantially more horizontal room.
3. The overlay has a left-side collapse/expand control.
4. Running a generated program collapses the Blockly overlay so the robot simulation is easier to watch.
5. Remove the Patrol-line-specific font-size workaround because the wider overlay is the layout-level fix.

- `apps/client/src/App.tsx`
- `apps/client/src/index.css`
