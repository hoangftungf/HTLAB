# C-005 — React shell + PixiJS renderer + simulation controls

status: done
deps: C-001
implements: FR1 (UI), FR5 (renderer)

## Scope

Build the React application shell and PixiJS 8 renderer that displays the simulation on screen.

1. **Monorepo setup** — Turborepo + pnpm workspaces: `packages/simulation-core`, `apps/client`
2. **React 18 + Vite + Tailwind CSS** — app shell with header, canvas area, sidebar placeholder
3. **PixiJS 8 renderer** — renders map image as background, robot as sprite/shape, trajectory as polyline, sensor indicators (5 small dots showing color intensity)
4. **Render loop** — requestAnimationFrame reads `sim.state` each frame, updates PixiJS display objects. Render FPS is independent of simulation tick rate.
5. **Zoom/pan** — mouse wheel zoom, drag to pan map, reset view button
6. **Robot sprite** — simple directional indicator (triangle or arrow) showing heading angle
7. **Trajectory line** — past path drawn as colored line (color = speed, or time gradient)

NOT in this card: telemetry panel UI (C-006), Blockly editor (C-007), state management wiring to Blockly.
