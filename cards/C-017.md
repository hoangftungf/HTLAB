# C-017 — Implement Whalesbot-style Create Function dialog for My Blocks

status: done
deps: C-016
implements: FR7

## Scope

Implement the Whalesbot Block Studio custom block creator ("Create a function" modal dialog) inside HTLAB, replacing the direct block spawning.

1. Rename the toolbox button to "Create new blocks" in `apps/client/src/blockly/toolbox.ts` and category styling to match Whalesbot.
2. In `apps/client/src/blockly/BlocklyEditor.tsx`, clicking "Create new blocks" opens a custom modal "Create a function - Whalesbot scratch".
3. The modal must include:
   - A live visual preview of the block: e.g. `[name_zgaO] [number1] [boolean1]` (based on current selections).
   - "add a numeric parameter" button (pill/capsule style).
   - "add a boolean parameter" button (hexagon style).
   - "add text labels" button.
   - Selected parameter details editor (Parameter Name, Default value) on the right side.
   - "OK" and "Cancel" buttons at the bottom.
4. When OK is clicked, create the `my_block_definition` block on the workspace with the configured name, parameter, type, and return type.
