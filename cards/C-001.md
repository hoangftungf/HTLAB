# C-001 — SimulationCore: robot kinematics + map loader + fixed timestep 60Hz

status: done
deps: none
implements: FR1

## Scope

Build the SimulationCore package: a zero-dependency TypeScript library that runs a deterministic 2D robotics simulation at fixed 60Hz timestep. This card covers:

1. **Differential drive kinematics** — update robot position (x, y, heading) each tick from left/right motor speeds, applying friction, acceleration limits, deadzone
2. **Map loader** — load a PNG map image + JSON metadata (dimensions, scale, startPose, checkpoints, finishZone), expose pixel data for sensor sampling
3. **Fixed timestep 60Hz** — `tick()` advances exactly 1/60s, physics is independent of render frame rate
4. **RobotConfig** — configurable wheelbase, wheelRadius, maxSpeed, maxAccel, friction, deadzone
5. **Test infrastructure** — vitest, test coverage > 80%, deterministic verification (same seed = same output)

Not in this card: sensor model (C-002), IR interpreter (C-003), telemetry (C-004), any UI code.
